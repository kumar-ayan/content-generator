# LinkedIn Architect | Frontend

The sleek, high-performance interface for the LinkedIn Architect platform. Built with React 19, Vite, and Tailwind CSS, featuring a blue-themed glassmorphic design optimized for content creation.

</p>

## 🚀 Features

- **LinkedIn-Centric Design:** Optimized for drafting and reviewing social posts.
- **Triple-Variation View:** Compare Professional, Storytelling, and Punchy drafts side-by-side.
- **Blueprint Archive:** Local history management for your best post ideas.
- **Micro-Interactions:** Smooth animations powered by Framer Motion.
- **One-Tap Copy:** Instant clipboard integration for seamless posting.

## 🛠️ Tech Stack

- **React 19:** Modern component architecture.
- **Vite:** Lightning-fast HMR and build times.
- **Tailwind CSS:** Custom design system with glassmorphism.
- **Framer Motion:** Fluid UI transitions and spatial awareness.
- **Lucide React:** Consistent LinkedIn-themed iconography.

## 📦 Installation

1. Navigate to directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

## 🏗️ Build

To generate a production build:
```bash
npm run build
```

The output will be in the `dist/` folder.
